Computes the grain size corresponding to a given cumulative percentage (DXX) for a reach with a specified grain size distribution.

Parameters
fi_rArray of fractions of material in each grain size class. For example, [0.15, 0.50, 0.35] represents 15%, 50%, and 35% of material in three grain size classes.
d_valueThe target DXX value (e.g., 50 for D50, 90 for D90) representing the grain size for which X% of the material is finer.
psiArray of grain sizes in negative log base-2 scale. For example, -3 represents 2³ = 8 mm, -4 represents 2⁴ = 16 mm.
Returns
Array of computed grain sizes (in meters) corresponding to the requested DXX for each input layer. If only one grain size class is present, the function returns that grain size. If d_value equals 100, it returns the smallest grain size.
  • The function supports both single-layer (fi_r as a vector) and multi-layer (fi_r as a matrix) inputs. For multi-layer cases, it processes each layer independently.
  • The function assumes that the provided grain sizes in psi are the maximum sizes of each class and not average sizes.
Note
The grain size distribution is interpolated in logarithmic (base-2) space, as is standard in sedimentology, and converted back to linear scale in meters.

45def D_finder(fi_r, d_value, psi):
46 """
47 @brief Computes the grain size corresponding to a given cumulative percentage (DXX)
48 for a reach with a specified grain size distribution.
49
50 @param fi_r Array of fractions of material in each grain size class.
51 For example, [0.15, 0.50, 0.35] represents 15%, 50%, and 35% of material
52 in three grain size classes.
53 @param d_value The target DXX value (e.g., 50 for D50, 90 for D90) representing the
54 grain size for which X% of the material is finer.
55 @param psi Array of grain sizes in negative log base-2 scale. For example,
56 `-3` represents 2³ = 8 mm, `-4` represents 2⁴ = 16 mm.
57
58 @return Array of computed grain sizes (in meters) corresponding to the requested DXX
59 for each input layer. If only one grain size class is present, the function
60 returns that grain size. If `d_value` equals 100, it returns the smallest
61 grain size.
62
63 @details
64 - The function supports both single-layer (`fi_r` as a vector) and multi-layer (`fi_r`
65 as a matrix) inputs. For multi-layer cases, it processes each layer independently.
66 - The function assumes that the provided grain sizes in `psi` are the maximum sizes
67 of each class and not average sizes.
68
69 @note The grain size distribution is interpolated in logarithmic (base-2) space,
70 as is standard in sedimentology, and converted back to linear scale in meters.
71
72 @example
73 ```
74 fi_r = [0.15, 0.50, 0.35]
75 d_value = 50
76 psi = [-3, -4, -5]
77 result = D_finder(fi_r, d_value, psi)
78 ```
79 """
80
81 dmi = np.power(2, -psi)/1000
82 nb_classes = len(psi)
83
84 # Handles the case of single layers input as vector and
85 # multiple layers input as matrices.
86 if fi_r.ndim == 1:
87 nb_layers = 1
88 fi_r = np.expand_dims(fi_r, axis=0)
89 else: # If multiple layers: AL: Do we need it? Neither Vjosa nor Po use it.
90 nb_layers = np.shape(fi_r)[0]
91
92 # If one class only, the target DXX value is the grain size of the class.
93 if dmi.size == 1:
94 return np.full(nb_layers, dmi[0])
95
96 # The D100 is the grain size of the biggest class.
97 elif d_value == 100:
98 return np.full(nb_layers, dmi[0])
99
100 else:
101 # Computes the inverse cumulative percentage of finer material,
102 # class by class, for each layer.
103 # Assumes that the given grain size is the maximum grain size of each
104 # class, and not the average grain size, for example.
105 perc_finer = np.empty(np.shape(fi_r))
106 perc_finer[:] = np.nan
107 perc_finer[:,0] = 100
108 for i in range(1, nb_classes):
109 perc_finer[:,i] = perc_finer[:,i-1] - fi_r[:,i-1] * 100
110
111 # Computes the target DXX value.
112 d_changes = np.zeros(nb_layers)
113 for k in range(nb_layers):
114 # Finds the class index of the percentage just above the target DXX value.
115 class_index = np.where(perc_finer[k, :] > d_value)[0].max()
116 # Ensure within valid range, which means that the interpolation to determine
117 # the target DXX value, if below the smallest material fraction, will be
118 # based on the adjacent above interval.
119 class_index = np.minimum(class_index, nb_classes - 2)
120
121 # Interpolation
122 perc_diff = perc_finer[k, class_index] - perc_finer[k, class_index + 1]
123 psi_diff = -psi[class_index] + psi[class_index + 1] # Because psi is negative...
124 # How much of the target cumulative percentage (d_value) lies above
125 # the percentage for class_index + 1, divided by the cumulative percentage
126 # difference to get a fraction.
127 interpolated_fraction = (d_value - perc_finer[k, class_index + 1]) / perc_diff
128 # Apply this fractional position to the grain size values and add the
129 # grain size values for class index + 1.
130 d_changes[k] = interpolated_fraction * psi_diff - psi[class_index + 1]
131
132 # Converts back to meters
133 d_changes[k] = np.power(2, d_changes[k]) / 1000
134 # Ensures no negative sizes by replacing them with the smallest size in dmi
135 d_changes[k] = d_changes[k] * (d_changes[k] > 0) + dmi[-1] * (d_changes[k] < 0)
136
137 return d_changes
